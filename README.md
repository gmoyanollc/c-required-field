# c-required-field
Simple client-side validation that emulates SharePoint `required-field` behavior.

Properties and methods are: 

  * tabId
  
  * labelId
  
  * setInputIsActive
  
  * setInputRequired
  
  * getInput
  
  * getInputIds
  
  * getLabel
  
  * hasInputValue
  
  * isInputActive
  
  * displayInputRequiredError
